URL commands:
1. set static bookmark: flyto.link?yourbookmark?www.thiswebsite.com
  flyto.link?yourbookmark will be directed to http://www.thiswebsite.com

2. set dynamic bookmark: flyto.link?yourbookmark?www.google.com/?search=[]
  flyto.link?yourbookmark?weather will be directed to http://www.google.com/?search=weather
